# Examples

The example collection is split into two categories.

- `simple`: small, focused CLIs that demonstrate one primary idea with a minimal setup
- `complex`: larger examples that combine multiple `quickli` features in one application

## Simple examples

- [Build a cat CLI with quickli](simple/cat-cli/README.md)
- [Build an ls CLI with quickli](simple/ls-cli/README.md)
- [Build a mkdir CLI with quickli](simple/mkdir-cli/README.md)
- [Build a head CLI with quickli](simple/quickhead/README.md)
